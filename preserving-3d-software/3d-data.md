# 3D Data

3D data forms an integral part of all 3D software. The most familiar is probably the 3D model, a digital object which encodes information about the representation of an object in 3D space. By interpreting the geometry and material data contained in the 3D model file, 3D software is able to construct a virtual representation of it.

From an XR preservation perspective, 3D data typically forms a part of larger set of source materials (e.g. a game engine project) which are used to build the 3D software. As detailed in [Preservation Strategies](preservation-strategies.md), maintaining long-term access to these source materials is not always feasible. It may therefore be desirable to preserve the 3D data independently of the project files — or at least understand what the implications of the formats used are. On this page you will find links to publications, presentations and initiatives/projects relating to the topic of 3D data preservation.
